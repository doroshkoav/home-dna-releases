# home-dna-releases

Журнал релизов проекта «Паспорт жилья». Исходный код — в приватном репозитории
[home-dna](https://github.com/doroshkoav/home-dna) (доступ по приглашению).

## Что здесь

- `CHANGELOG.md` — история версий в формате [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).
- GitHub Releases этого репозитория — сборки (APK/AAB, ссылки на EAS-билды) и релизные заметки для каждой версии, когда появятся первые сборки для тестирования.

## Что не здесь

Исходный код, issues по разработке, техническая документация (архитектура, план) — всё
это остаётся в `home-dna`. Здесь фиксируется только то, что было отдано во внешнее
тестирование или в прод: что вышло, когда и что изменилось.

## Версионирование

[SemVer](https://semver.org/lang/ru/). Пока проект не вышел даже в закрытое тестирование
(см. `plan.md` в основном репозитории, неделя 15) — версии `0.x.y`, релизы не гарантируют
обратной совместимости.
