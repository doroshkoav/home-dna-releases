## [Unreleased]

### Готово, но не выпущено
- Backend: модули-заглушки, схема БД, воркер `ai-pipeline` — без реализованной бизнес-логики.

## [0.1.0] - 2026-07-08

Первая сборка — debug-APK для внутреннего теста на устройстве. Backend не подключён,
все данные локальные моки (см. `apps/mobile/src/context/AppState.tsx` в `home-dna`).

### Добавлено
- 6 экранов мобильного приложения по прототипу (`design/prototype`): Онбординг, Дашборд,
  Документы, Гарантии, Журнал, Календарь — с состояниями пусто/загрузка/заполнено/ошибка.
- Тёплая цветовая тема (светлая/тёмная) на основе `packages/design-tokens`.

### Известные ограничения
- Debug-сборка, не подписана для публикации в маркеты.
- Собрана только под arm64-v8a (наиболее распространённая архитектура) — armeabi-v7a/x86/x86_64 не включены в этот билд.
- Backend не подключён — все действия (загрузка документов, гарантий и т.д.) работают на локальном моке, без сохранения на сервере.
- iOS-сборки нет.
